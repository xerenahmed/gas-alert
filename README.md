# Gas Alert

## Environmet variables

### BELOW_GAS
If the gas drops below this value, you will be notified.

### ETHERSCAN_API_KEY
API key from EtherScan.io

### SENDGRID_API_KEY
API key from SendGrid. SendGrid provides a cloud-based service that assists businesses with email deliver

### SENDGRID_FROM_EMAIL
Email you set in SendGrid.
